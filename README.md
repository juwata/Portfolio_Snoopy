# Portfólio Colaborativo 

Este projeto foi desenvolvido como parte de um trabalho acadêmico com o objetivo de
praticar **Git, GitHub e desenvolvimento web**, além de integrar páginas pessoais de
cada integrante em um único site de portfólio.

O site reúne informações sobre os integrantes do grupo e seus perfis, tudo organizado
em uma interface única.

---

## Objetivos do Projeto

- Praticar versionamento de código com **Git e GitHub**
- Trabalhar de forma colaborativa em um mesmo repositório
- Criar páginas pessoais (portfólios individuais)
- Integrar todas as páginas em um único site
- Aplicar conceitos básicos de HTML e CSS
